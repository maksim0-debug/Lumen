# ⚡ Lumen

Сучасний додаток для моніторингу відключень електроенергії для Київської області (ДТЕК Київські регіональні електромережі).
Працює на **Android** та **Windows**.

## ✨ Можливості
- 📈 **Потужна аналітика:** Детальна статистика відключень та особисті інсайти:
  - **Дашборд:** Зведена статистика за день/тиждень/місяць, середня тривалість відключень та визначення "найгірших" днів тижня.
  - **Оцінка точності графіків:** Порівняння реальної наявності світла з офіційним графіком ДТЕК (розрахунок відхилень та затримок перемикання).
  - **Рекорди та теплові карти:** Візуалізація щоденних трендів, теплова карта відключень та фіксація ваших рекордів (найдовші періоди зі світлом та без).
  - **Порівняння груп:** Зручне зіставлення ситуації з відключеннями між різними чергами за обраний період.
   - **Вплив на життя:** Підрахунок втраченого робочого часу (09:00–18:00) та зіпсованих вечорів (19:00–23:00).
   <img width="479" height="817" alt="image" src="https://github.com/user-attachments/assets/4f1c50d6-9241-4ab6-930a-15ff94095e2c" />
- 🔌 **Моніторинг реальних відключень:** Можливість підключити та налаштувати власний датчик наявності світла. Додаток отримує ці дані через Firebase, що дозволяє в реальному часі бачити фактичний статус електромережі вдома.
- 📈 **Потужна аналітика та порівняння:** Аналіз відключень на основі зібраних реальних даних:
  - **Реальність vs Графік:** Порівняння фактичних відключень з офіційним розкладом ДТЕК для оцінки точності графіків та розрахунку затримок включення/виключення.
- 🚀 **Кросплатформеність:** Єдиний код для мобільної та десктопної версії.
- 🕒 **Таймер зворотного відліку:** Показує точний час до ввімкнення або вимкнення світла.
- 📊 **Наочний графік:** Візуалізація розкладу.
- 🔔 **Розумні сповіщення:**
  - Нагадування за 1 годину, 30 хвилин та 5 хвилин.
  - Окремі налаштування для ввімкнення та вимкнення.
  - Сповіщення про зміну графіку
  - Можливість ввімкнути сповіщення для кількох черг одночасно.
- 📱 **Віджети (Android):** Підтримка декількох віджетів для різних груп одночасно.
- ⏳ **Збереження змін графіку** Зберігається кожна версія графіка.
